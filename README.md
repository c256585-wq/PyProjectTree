# PyProjectTree
Построение бинарного дерева по заданным кодам путей (Вариант 19)

Для запуска программы необходимо извлечь архив и запустить main.py 

✅ ВНУТРИ есть ГОТОВЫЕ TXT файлы для проверки! ✅

small.txt - маленькое дерево  
big.txt - большое дерево  
7.txt - повторяющийся код (узел)  
8.txt - некорректное значение узла дерева  
9.txt - некорректный код узла дерева (не 0 или 1)  
10.txt - больше или меньше 2 значений в строке файла  
11.txt - проверка влияют ли пустые строки в файле  
12.txt - проверка отображения пустых промежуточных узлов  
13.txt - самое минимальное дерево  

Отдельно можно проверить, что ловятся ошибки при: пустом файле; отсутствии файла; попытке вывести дерево без загруженного файла; некорректном вводе в меню.
